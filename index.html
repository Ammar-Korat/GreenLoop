# GreenLoop 🌿 A recycling app for Egypt — Close the Loop. Recycle Smarter. ## Project Structure ``` greenloop/ ├── index.html ← Main HTML (all screens & markup) ├── css/ │ └── styles.css ← All styles & CSS variables ├── js/ │ └── app.js ← All JavaScript logic & data └── README.md ``` ## Files Overview | File | Purpose | |------|---------| | `index.html` | All screen markup: loading, survey, auth, dashboard, pickup, payment, confirm, impact, support, nav | | `css/styles.css` | CSS variables, layout, components, animations, responsive rules | | `js/app.js` | Materials data, routing (`goTo`), survey logic, auth tabs, upload handler, materials grid, pickup/payment toggles, FAQ, toast | ## How to Run Just open `index.html` in any modern browser. No build step required.
